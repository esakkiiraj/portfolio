Blinkit Sales & Operations Dashboard

[Project link]

I recently completed a comprehensive data analysis project on Blinkit, one of India’s leading quick-commerce platforms. The goal was to extract meaningful insights from real-world order and delivery data using Power BI. This project involved building a fully interactive dashboard with KPIs such as Total Revenue, Order Volume, Average Delivery Time, and Product Category Performance. I was responsible for cleaning and transforming the dataset using Power Query, calculating custom measures using DAX, and designing the user interface to support drill-through analytics and slicers for better decision-making. Identified peak demand hours, high-performing product zones, and delivery inefficiencies to improve operational planning and business outcomes.
Tech Stack : Power BI | Database : SQL | ETL Tool : Power Query

Ola Ride Data Analysis Dashboard

[Project link]

I recently developed a Power BI dashboard to analyze Ola’s ride-sharing data, with a specific focus on identifying peak booking hours and understanding cancellation patterns. Using Excel for data cleaning and Power Query for transformation, I modeled key fields to extract ride timestamps, trip durations, and cancellation flags. The dashboard highlights KPIs such as Peak Demand Time Slots, Total Rides, Cancellation Rate (by user and driver), and Zone-Wise Ride Distribution. Visualizations enable stakeholders to monitor hourly traffic, optimize fleet allocation, and reduce trip dropouts. This project strengthened my ability to work with real-world mobility data and uncover time-based behavior trends through insightful dashboards.
Tech Stack : Power BI | Database : SQL | ETL Tool : Power Query | Source : Excel

Electronics Sales Performance Dashboard

[Project link]

I recently worked on an end-to-end sales analysis project for an electronics retail dataset. This project aimed to identify key sales trends, top-performing products, regional performance, and seasonal demand spikes. I cleaned and structured raw transactional data in Excel, then used Power Query in Power BI for data transformation and modeling. The dashboard highlights KPIs such as Total Revenue, Units Sold, Average Profit Margin, and Product Category Contribution. Drill-through and dynamic filters were implemented to allow users to analyze performance across regions and time frames. This project provided real-world exposure to analyzing commercial datasets and presenting insights in a business-friendly format.
Tech Stack : Power BI | Database : SQL | ETL Tool : Power Query | Source : Excel
